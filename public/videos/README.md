# Beach Video Background

To use a beach video background:

1. Download a free beach/ocean video (MP4 format recommended)
2. Place it in this directory and name it `beach.mp4`
3. Recommended sources:
   - Pexels: https://www.pexels.com/search/videos/beach/
   - Pixabay: https://pixabay.com/videos/search/beach/
   - Unsplash: https://unsplash.com/s/videos/beach

The video will automatically be used as the background on the sign-in page.
